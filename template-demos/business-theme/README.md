# Business Theme - Demo Template

Este é um tema de demonstração para o CMS Nextjs.

## Estrutura

- `pages/` - Páginas do tema (index.js, about.js)
- `components/` - Componentes reutilizáveis (Hero, Features, Contact)

## Compatibilidade

Cada componente usa atributos `data-block-type` e `data-block-id` para identificação no CMS.

## Como Importar

1. Vá em Templates → Importar Pasta
2. Selecione a pasta `business-theme`
3. O tema será importado com todas as páginas

## Blocos Inclusos

- Hero (título + parágrafo + imagem)
- Features (grid de recursos)
- Contact (seção de contato)
- About Hero (página sobre)
- Mission (missão da empresa)
